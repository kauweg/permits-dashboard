Static-site setup for the easiest GitHub → Render path.

Why this version is better

  • Cleaner executive framing instead of a raw hobby dashboard look.
  • Seattle and Bellevue load together through live public endpoints.
  • Fallback demo dataset lets the page open even if a public API hiccups.

  • Push repo to GitHub.
  • Create a Render Static Site.
